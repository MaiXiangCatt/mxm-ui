Dropdown 下拉菜单 ​

将动作或菜单折叠到下拉菜单中。

基础用法 ​

通过组件 slot 来设置下拉触发的元素以及需要通过具名 slot 为 dropdown 来设置下拉菜单。

通过 slot 传入 item
通过 props 传入 item

<template>
  <div class="row">
    <div class="col">
      <div class="desc">通过 slot 传入 item</div>
      <mxm-dropdown>
        <span class="dropdown-link">
          Dropdown List
          <mxm-icon icon="angle-down" />
        </span>
        <template #dropdown>
          <mxm-dropdown-item command="1">Action 1</mxm-dropdown-item>
          <mxm-dropdown-item command="2" label="Action 2" />
          <mxm-dropdown-item command="3" disabled>Action 3</mxm-dropdown-item>
          <mxm-dropdown-item command="4" divided>Action 4</mxm-dropdown-item>
        </template>
      </mxm-dropdown>
    </div>
    <div class="col">
      <div class="desc">通过 props 传入 item</div>
      <mxm-dropdown :items="items">
        <span class="dropdown-link">
          Dropdown List
          <mxm-icon icon="angle-down" />
        </span>
      </mxm-dropdown>
    </div>
  </div>
</template>

<script setup lang="ts">
import type{ DropdownItemProps } from "mxm-ui";

const items: DropdownItemProps[] = [
  { command: "1", label: "Action 1" },
  { command: "2", label: "Action 2" },
  { command: "3", label: "Action 3", disabled: true },
  { command: "4", label: "Action 4", divided: true },
];
</script>

<style scoped>
.row {
  display: flex;
  flex-wrap: wrap;
}
.col {
  flex: 1;
}
.desc {
  display: block;
  color: var(--mxm-text-color-secondary);
  font-size: 14px;
  margin-bottom: 20px;
}
.dropdown-link {
  display: flex;
  align-items: center;
  cursor: pointer;
  color: var(--mxm-color-primary);
  i {
    margin-left: 8px;
  }
}
</style>

点击菜单隐藏 ​

hide-on-click="true"(default)
hide-on-click="false"

指令事件 ​

点击菜单项后会触发事件，用户可以通过相应的菜单项 command 进行不同的操作。

<template>
  <mxm-dropdown :items="items" @command="handleCommand">
    <span class="dropdown-link">
      Dropdown List
      <mxm-icon icon="angle-down" />
    </span>
  </mxm-dropdown>
</template>

<script setup lang="ts">
import {
  type DropdownItemProps,
  type DropdownCommand,
  MxmMessage,
} from "mxm-ui";

const items: DropdownItemProps[] = [
  { command: "1", label: "Action 1" },
  { command: "2", label: "Action 2" },
  { command: "3", label: "Action 3", disabled: true },
  { command: "4", label: "Action 4", divided: true },
];

function handleCommand(command: DropdownCommand) {
  MxmMessage.info("click on item " + command);
}
</script>

<style scoped>
.dropdown-link {
  display: flex;
  align-items: center;
  cursor: pointer;
  color: var(--mxm-color-primary);

  i {
    margin-left: 8px;
  }
}
</style>

下拉方法 ​

可以手动使用 DropdownInstance.open 或 DropdownInstance.close 以打开或关闭下拉菜单


<template>
  <mxm-button @click="() => dropdownRef?.open()">open</mxm-button>
  <mxm-button @click="() => dropdownRef?.close()">close</mxm-button>
  <br />
  <mxm-dropdown ref="dropdownRef" :items="items" manual>
    <span class="dropdown-link">
      Dropdown List
      <er-icon icon="angle-down" />
    </span>
  </mxm-dropdown>
</template>

<script setup lang="ts">
import type { DropdownItemProps, DropdownInstance } from "mxm-ui";
import { ref } from "vue";

const items: DropdownItemProps[] = [
  { command: "1", label: "Action 1" },
  { command: "2", label: "Action 2" },
  { command: "3", label: "Action 3", disabled: true },
  { command: "4", label: "Action 4", divided: true },
];
const dropdownRef = ref<DropdownInstance>();
</script>

<style scoped>
.dropdown-link {
  display: flex;
  align-items: center;
  cursor: pointer;
  color: var(--mxm-color-primary);
  margin-top: 20px;
  i {
    margin-left: 8px;
  }
}
</style>

触发对象 ​

设置 split-button 属性来让触发下拉元素呈现为按钮组，左边是功能按钮，右边是触发下拉菜单的按钮。

<template>
  <div class="row">
    <mxm-dropdown :items="items">
      <mxm-button type="primary">
        Dropdown List
        <mxm-icon icon="angle-down" style="margin-left: 8px" />
      </mxm-button>
    </mxm-dropdown>
    <mxm-dropdown
      :items="items"
      type="primary"
      @click="handleClick"
      split-button
    >
      Dropdown List
    </mxm-dropdown>
  </div>
</template>


<script setup lang="ts">
import { type DropdownItemProps, MxmMessage } from "mxm-ui";

const items: DropdownItemProps[] = [
  { command: "1", label: "Action 1" },
  { command: "2", label: "Action 2" },
  { command: "3", label: "Action 3", disabled: true },
  { command: "4", label: "Action 4", divided: true },
];

function handleClick() {
  MxmMessage.info("button click");
}
</script>

<style scoped>
.row div {
  margin-right: 16px;
}
</style>

Dropdown API ​

Dropdown Props ​

属性名说明类型Default
type菜单按钮类型同 Button 组件 的 type--
size菜单尺寸同 Button 组件 的 size--
split-button下拉触发元素呈现为按钮booleanfalse
disabled是否禁用booleanfalse
trigger触发方式'hover' | 'click' | 'contextmenu'hover
placement弹出位置'top' | 'top-start' | 'top-end' | 'bottom' | 'bottom-start' | 'bottom-end' |...bottom
hideOnClick点击菜单项时是否隐藏下拉菜单booleantrue
show-timeout展开下拉菜单的延时,仅在 trigger 为 hover 时有效number150
hide-timeout收起下拉菜单的延时,仅在 trigger 为 hover 时有效number150

Dropdown Events ​

事件名说明类型
clicksplit-buttton 为 true 时,点击左侧按钮触发(event: MouseEvent) => void
visible-change下拉菜单显示或隐藏时触发(visible: boolean) => void
command点击菜单项时触发(command: string| number) => void

Dropdown Slots ​

插槽名说明子标签
default默认插槽-
dropdown下拉菜单DropdownItem

Dropdown Expose ​

方法名说明Type
open打开下拉菜单() => void
close关闭下拉菜单() => void

DropdownItem Props ​

属性名说明类型Default
command菜单项指令string | number-
label菜单项文本string-
disabled禁用状态booleanfalse
divider添加分割线booleanfalse
